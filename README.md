# Cowork Cleaning Admin

共享空間清潔服務 — 供應商管理後台系統。

## 簡介

本專案是一套供應商管理後台，用於管理共享空間的清潔服務流程，包含：

- 訂單管理（建立、追蹤、狀態更新）
- 供應商資料管理
- 商品與服務分類
- 利潤與佣金計算
- 報表匯出與列印

技術架構：React 18 + Tailwind CSS，以單一 HTML 檔案部署，無需後端伺服器。

## 安裝方式

### 環境需求

- 任意現代瀏覽器（Chrome、Firefox、Safari、Edge）
- Git

### 步驟

1. Clone 專案：

```bash
git clone git@github.com:njnjshoooo/cowork-cleaning-admin.git
cd cowork-cleaning-admin
```

2. 開啟後台頁面：

```bash
open 供應商管理後台.html
```

或直接在瀏覽器中開啟 `供應商管理後台.html` 即可使用。
